Taxomplete

A widget to select a biological using [Plazi](http://plazi.org/)'s open taxonomical literature an [Awesomplete](https://leaverou.github.io/awesomplete/)'s awesome autocomplete.

In the following form the third field has taxomplete enabled, type a couple of characters to see the difference.




How does it work?

The Taxomplete widget executes SPARQL queries to the [LINDAS Linked Data Service](https://lindas-data.ch/) to query for species in the Plazi dataset, while you type. Taxomplete is developed by [FactsMission](https://factsmission.com/) and available on [Github](https://github.com/plazi/taxomplete/tree/master).
