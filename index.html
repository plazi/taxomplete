Taxomplete

A widget to select a biological using Plazi's open taxonomical literature an Awesomplete's awesome autocomplete.

In the following form the third field has taxomplete enabled, type a couple of characters to see the difference.




How does it work?

The Taxomplete widget executes SPARQL queries to the LINDAS Linked Data Service to query for species in the Plazi dataset, while you type. Taxomplete is developed by FactsMission and available on Github.
